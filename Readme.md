 Juego de Lucha en Java
[![Java CI with Maven](https://github.com/Juansanchezzz/juego-lucha-patrones/actions/workflows/ci.yml/badge.svg)](https://github.com/Juansanchezzz/juego-lucha-patrones/actions/workflows/ci.yml)


Proyecto desarrollado en Java utilizando Maven, JUnit 5, Mockito y patrones de diseño.

 Características

- Sistema de pelea entre personajes
- Uso de patrones de diseño
- Pruebas unitarias
- Integración continua con GitHub Actions
- Reportes de cobertura con JaCoCo

 Patrones implementados

 Factory Method
Permite crear distintos tipos de personajes.

 Strategy
Permite cambiar estrategias de ataque.

 Decorator
Permite agregar armas o habilidades especiales.

 Tecnologías utilizadas

- Java 17
- Maven
- JUnit 5
- Mockito
- JaCoCo
- GitHub Actions

## Ejecutar el proyecto

Compilar:

mvn compile
